# Desk Buddy Robot

Desk Buddy is a small autonomous desk companion robot built around an STM32F411 microcontroller. The robot uses VL53L0X Time of Flight sensors to avoid falling off a desk, DC motors with encoders for movement, servo-driven llama ears for expressive behavior, and an SSD1306 OLED display for facial animations.
