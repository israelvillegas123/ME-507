void task_distance_initialize ( void )

Initializes the VL53L0X distance sensor task.

Scans the I2C bus, initializes both ToF sensors, and reports startup status through USB debug messages.

◆ TOF_InitTwo()

int16_t TOF_InitTwo ( void )

Initializes both VL53L0X sensors.

Uses the XSHUT pins to turn on each sensor individually and assign unique I2C addresses.

Returns
0 if both sensors initialize successfully, otherwise a VL53L0X error code.
